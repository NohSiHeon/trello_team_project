## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```

## Commit Convention

| 작업 타입 | 작업내용                    |
| --------- | --------------------------- |
| feat      | 새로운 기능 추가            |
| fix       | 버그 수정                   |
| perf      | 성능 개선                   |
| build     | 모듈, 패키지 설치, 삭제     |
| add       | 새 파일 생성                |
| move      | 파일 옮김, 정리             |
| remove    | 파일 삭제                   |
| rename    | 파일명, 폴더명, 변수명 수정 |
| refactor  | 코드 리팩토링               |
| chore     | 그 외 자잘한 수정           |
| comment   | 주석 추가, 수정             |
| test      | 테스트 코드 추가, 수정      |
| docs      | readme 문서 수정            |
| gitfix    | gitignore 수정              |
